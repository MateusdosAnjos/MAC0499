Introdução

Phoenix Rising é um jogo no estilo puzzle cujo objetivo é facilitar o entendimento de conceitos de programação.

O jogador deverá pensar em como organizar os comandos fornecidos em cada nível de forma a conseguir a saída desejada, para isso deverá compreender o que cada comando executa e como é feita a computação. Para auxiliá-lo o jogo fornece ajuda explicando, sempre que solicitado, o funcionamento do comando, permitindo que o jogador fixe essas informações enquanto joga.

A jogabilidade é a mais simples possível, portanto o jogo não necessita de nenhuma habilidade em games, apenas o raciocínio lógico.

Além disso, caso alguém queira inserir novos comandos no jogo, a estrutura do código fonte permite uma rápida inserção de tais comandos, fazendo com que o jogo possa evoluir de maneira versátil atendendo diversos públicos.
