Resumo

Novas tecnologias surgem a cada dia permitindo o desenvolvimento de novas aplicações que facilitem a vida dos usuários. Entretanto uma parte das pessoas não consegue se beneficiar totalmente dessas facilidades, pois a acessibilidade dos aplicativos ainda é diminuta.

Desta forma este trabalho busca explorar as possíveis melhorias em uma extensão para browser que possibilita a navegação utilizando o olhar, bem como analisar as dificuldades e propor alguma forma de aborda-las.

Para isso devemos entender um pouco mais do abrangente conceito de interação humano computador, propor novas formas de captar dados do dispositivo de eye-tracker, modificar a forma de exibir uma página web e fazer com que a extensão seja moldada a fim de suprir as necessidades do usuário para que ele tenha uma experiência exitosa dentro de seu contexto de uso.
