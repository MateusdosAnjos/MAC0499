Resumo dos Trabalhos (Proposta)

A extensão será desenvolvida para Google Chrome, pois é o browser mais utilizado atualmente.

Devemos estudar um pouco da história dos dispositivos de eye-tracker relacionando com o avanço da inclusão digital.

Estudar como o dispositivo eye-tracker escolhido funciona (ainda devemos escolher o dispositivo).

Estudar como as soluções existentes abordaram o problema para encontrar possíveis caminhos a seguir a fim de fazer alterações e obter melhorias.

Aprender a criar extensão para google chrome para julgar de maneira mais precisa o que será possível implementar neste trabalho.

Propor alguma melhora na captação da informação do olhar e/ou no modo como exibir o conteúdo da página web a fim de melhorar a experiência do usuário dentro de seu contexto de uso.

Tentar minimizar o stress do usuário ao utilizar a extensão, para que, mesmo com a deficiência, ele consiga, de forma satisfatória, trocar informação durante sua interação com a máquina.

Estar aberto a novas ideias e possíveis mudanças.
